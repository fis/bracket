Detailed Description

template<typename Enum, typename T, int N>
struct base::EnumArray< Enum, T, N >

An array for mapping (dense) enumeration constants to a type.

This type is expected to be used in constexpr declarations, in which case it will turn into the raw bytes of an array of T of length N.

N should be one more than the highest-valued constant of Enum. Including higher values in the array initializer will be detected at compile time, but looking them up will not.
