Detailed Description

template<typename T, typename C, void(C::*)(std::unique_ptr< T >) method>
class event::ClientPtr< T, C, method >

Callback member function adapter for Client (with pointer data).

This class is a convenience wrapper for client events. Constructing it automatically registers a client event on a provided Loop. The object can be called, which (indirectly) causes the specified callback function to be called with the same data.
