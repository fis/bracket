Detailed Description

template<typename T>
class base::owner_set< T >

Set of ownership-holding pointers.

base::owner_set<T> has more or less the semantics of std::unordered_set<std::unique_ptr<T>>, with the exception that it allows for deleting items based on a raw pointer to the item. It also only implements some of the container methods.
