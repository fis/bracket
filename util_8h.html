std::unique_ptr< google::protobuf::io::FileOutputStream > proto::OpenFileOutputStream ( const char *  path)

Opens a file descriptor and wraps it into a protobuf output stream.

No access to the file descriptor is provided, and it will be automatically closed when the stream is destroyed.

The file descriptor is opened with the O_CREAT and O_APPEND flags, with mode 0644 (modified by umask).
