enum base::LogLevel

Logging levels.

Enumerator
VERBOSE 

Verbose logging, usually not even needed for debugging.

Not available in optimized builds.

DEBUG 

Debug logging.

Not available in optimized builds.

INFO 

Informational messages that do not necessarily mean anything unexpected has happened.

WARNING 

Warning messages of slightly unusual activities.

ERROR 

Potentially serious error conditions.

FATAL 

Invariably fatal errors.

Only used if the program will throw an exception.
